# Algorithm-for-File-Updates-in-Python
I created an algorithm that removes IP addresses listed in the remove_list variable from the allow_list.txt file of approved IP addresses used to view restricted content.  This involved opening the file, converting it into a string and a list, and using a for loop to iterate over the IP addresses in the remove list. In each iteration, it evaluated if the element was in the ip_addresses, and if it was, I used the .remove method to remove it.  Then I used the .join method to convert ip_addresses back into a string so that I could update the contents of the allow_list.txt file.
